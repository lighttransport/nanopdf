# StandardEncoding Regression Samples

Real-world PDFs that exercise text extraction with StandardEncoding and `/Differences` dictionaries.  All files were downloaded on $(date -u +%Y-%m-%d) from the following sources:

- `pdfa_font_encoding.pdf` — [PDF Association](https://pdfa.org/wp-content/uploads/2018/06/1530_Seggern.pdf)
- `sas_encoding_journey.pdf` — [SAS Global Forum 2020 Proceedings](https://support.sas.com/resources/papers/proceedings20/4561-2020.pdf)
- `nist_traditional_encoding.pdf` — [NIST Traditional Encoding Guidance](https://www.nist.gov/system/files/documents/2021/03/23/ansi-nist_2010_traditional_encoding.pdf)

These documents are intended for parser regression only; do not redistribute outside the repository.
